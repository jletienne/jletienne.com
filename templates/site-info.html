What is the point of this site

The site was built with Python and Flask. It's based off of The Flask Mega-Tutorial by Miguel Grinberg. Web hosting is done on Heroku, while the code is hosted on GitHub for version control.
I get it you can make a much better site on WordPress or Squarespace. That's fine with me, but to make something truly unique you need to build it yourself. That's why I didn't use Django. Ideally the site develops as I learn new skills. You're looking at a Framework. How can I design a cohesive experience on the internet? How can I architect information in a clean and consistent manner? HOw can I to manage templates to reduce redunancy and create dynamic pages? And most importantly how can I improve the skills I have to create something truly worthwhile.
